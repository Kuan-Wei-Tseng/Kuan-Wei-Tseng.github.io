Selected Coursework

License Plate Detection and Recognition

We propose an intelligent license plate detection and recognition system which supports illuminance-invariant license plates detection and optical character recognition (OCR). The input image is sent to the detection module, where we apply cross power spectrum to find the top-k candidate patches that are similar with pre-computed template, which is a set of eigenvectors extracted from numerous license plate. Next, we reject outliers by physical constraint of Taiwanese license plate and perform OCR by similarity estimation with training data to recognize the license plate numbers.
