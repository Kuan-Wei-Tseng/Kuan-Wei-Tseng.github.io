Visual-Inertial Fusion for Object Pose Tracking

Sensor Fusion, 3D Computer Vision, Robotics

Object pose estimation and tracking is to find the relative 6DoF transformation, including translation and rotation, between the object and the camera. We present a visual-inertial fusion framework for object pose estimation and tracking which supports flexible framerate and is able to perform in real-time on mobile devices.

Deep3D: 3D VIDEO STABILIZATION WITH DEEP LEARNING

3D Computer Vision, Deep Learning, Image Processing

3D Video Stabilization aims to remove shakey and jittery contents from videoes using estimated 3D information. We propose Deep3D Stabilizer, a novel 3D depth-based learning method for video stabilization, which took advantage of the self-supervised framework on jointly learning depth and camera ego-motion estimation.

aBiO: ACTIVE Bi-OLFACTORY DISPLAY FOR VR

Human Computer Interaction, Virtual Reality

Including olfactory cues in virtual reality (VR) would enhance user immersion in the virtual environment. We present an active bi-olfactory display system that delivers scents precisely to specific locations. aBio provides users with a natural olfactory experience in free air by colliding two vortex rings launched from dual speaker-based vortex generators.

PixStabNet: FAST DEEP ONLINE VIDEO STABILIZATION

Computer Vision, Deep Learning, Image Processing

We propose a multi-scale convolutional neural network (PixStabNet) which stabilizes each unstable frame in real time without using future frames. We estimate a pixel-based warping map to transform each pixel for more precise modelling. Also, we propose well-designed loss functions along with a two-stage training scheme to enhance network robustness.

AUGMENTED TAI-CHI CHUAN PRACTICE TOOL

Research Project (MIPR 2021)

We propose an augmented reality Tai Chi Chuan (TCC) practice tool with pose evaluation to help people practice TCC on their own. The tool consists of an optical see-through head-mounted display, external cameras, digital compasses, and a server. Users learn TCC movements from virtual coaches in AR and check their postures with pose evaluation module.
